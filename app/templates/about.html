News Feather - About

News Feather is a news aggregator site that displays all the recent news from sources (i.e. CNN, Wired, TC, etc.).

The site was developed by one person, Mattias Zurkovic, because I wanted it to help me learn Python web development and using the lightweight Flask framework.

News Feather is also open-source! While under development I tracked all the files, changes and code on GitHub. If you are familiar with Git and GitHub go ahead and get the News Feather source code, if not read their tutorials and documentation.
